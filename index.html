How to use Google Fonts?

Use Google Fonts

  1. Access fonts.google.com.
  2. Type in a font name inside the search bar.
  3. Click on the name of the font to show a list os styles vaiables to import.
  4. On the side of each style you would like to import, click on the + to select the style.
  5. On the right handside panel, select @import at the Use o the web section.
